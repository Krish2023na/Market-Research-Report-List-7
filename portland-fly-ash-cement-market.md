<p><strong>The Portland Fly Ash Cement Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Cement with Class F Fly Ash</li><li>Cement with Class C Fly Ash</li></ul></p>
<p><p>The Portland Fly Ash Cement market includes two primary types based on the classification of fly ash: Class F and Class C. Cement with Class F fly ash is typically derived from burning anthracite or bituminous coal, offering enhanced durability and resistance to sulfate attack. In contrast, cement with Class C fly ash is produced from lignite coal, containing more calcium, which allows for faster strength gain and improved workability. Both types improve sustainability by leveraging industrial by-products in cement production.</p></p>
